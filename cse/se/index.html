Ad-Hoc Lego Network

What is it?

The central server (a laptop) is the information sink. It can be
visualized as data collection equipment, which issues commands
to the legos to report various details about its surrounding. Here we just
spread the legos in a room and the functions of the lego are sensing and
relaying. Some legos may not be within the range of the server in which
case the communication takes place through nearby legos. For example, the
central server may issue a command "GET THE LIGHT INTENSITY". The command
will be relayed to alll the legos and the legos respond with the light
intensity values at that instant. Similarly we can have different commands
like "GET THE CURRENT ENERGY REMAINING IN THE BATTERY" etc.
