  Tcl_Interp * interp, 
  int  argc, 
  char ** argv 
)
Givben an FSM, sorts the transition relation array (tau array) of the FSM according to the await-dependency relation and conjoins the taus till the as input is reached. It then returns the FSM with the new tau array of the conjoined mdds (instead of the old one). (So for FSMs with many small taus, we get fewer larger taus making the computation on them more efficient.

Side Effects the updateBdd of the FSM is modified.
