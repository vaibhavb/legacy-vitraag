Memory Protection

The essential point is that each domain must have its own address space for enabling protection [Lam71]. In absence of paging and segmentation each domain will have to have private memory.

Palm has no protection mechnanism against flash memory attack owing to its memory design [Mud01]. However the entire system data can be kept in an encrypted form to avoid maligancy [Res01].
