Password Retrieval

The users of portable devices have no keyboard and require character input with a pen, so many times the users have very shortpassowrds. Convenience gives way to gulliblity. In PALM password obfuscation is weak, it gives way to XOR attack[Mud01]. Password mechanism can be strenghtened by:
  • Challenge/ Response mechanism on the network.
  • Encrypt and Salt Credentials stored on the system.
  • Implementation of power-on passsword.
  • Alternative passwords like signatures, graphical passwords.
