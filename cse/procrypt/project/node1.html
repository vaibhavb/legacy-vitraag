Why this?

Embedded devices form the current computing trend. Handheld devices with protability, easy access and network ability are increasingly gaining computing share. With many people using these devices, the security aspect of them warrant a more critical study. Handhelds typically have no security framework. Its not possible to build a secure application on top of an insecure one.
