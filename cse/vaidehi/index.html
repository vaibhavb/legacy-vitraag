What did Digital Design Teach?

1. Digital logic and or not nand xor.
2. How to combine this gates to make an interesting circuit.
3. Use minimum of gates by K-map method.
4. Doing a hierarchical and simple design by using things like multiplexers.
5. Proceeding to complex things like adder, multipler, ALU and then the microprocessor!!
In all this the idea of LOGIC remains same only TOOLS change. To solve a COMPUTATIONAL problem one requires good digital tools which can be glued in a meanfulway. Edit
