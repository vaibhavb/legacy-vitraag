
  state_print - Prints the values of all variables in a state.
     _________________________________________________________________
   
   state_print [-h] <state_name>
   
   The command prints the values of all variables in a state. The command
   takes as argument a state name generated by the commands enum_init or
   enum_post. Command Options:
   
   -h
          Print usage.
