The mdl package

Building Modules and Performing Various Operations.

By Freddy Mang

This package contains structures and routines related to modules. It serves two main purposes: 1. providing module structures for storing atoms, variables and invariants; 2. providing routines for various module level operations, such as parallel composition, variable hiding and renaming.
