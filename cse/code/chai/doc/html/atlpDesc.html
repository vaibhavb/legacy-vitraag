The atlp package

Routines for parsing, writing and accessing ATL formulas.

By Freddy Mang

This package implements a parser for ATL (Alternating-time Temporal Logic) formulas. ATL is a language used to describe properties of systems. For the syntax of ATL formulas, refer to the ATL syntax manual.
