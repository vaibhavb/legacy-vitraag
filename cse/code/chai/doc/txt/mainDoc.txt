The main package

External header file for the main package.

Shaz Qadeer

**********************************************************************

Main_AppendElement()           Append element to the Tcl interpreter.

Main_AppendResult()            Append result to the Tcl interpreter.

Main_ClearResult()             Clears the result in the Tcl interpreter.

Main_ConvertIntToString()      Convert an integer into string.

Main_End()                     Ends the main package.

Main_Init()                    Initializes the main package.

Main_ManagerReadMddManager()   Returns the mdd manager.

Main_ManagerReadModuleManager()
                               Returns the module manager.

Main_ManagerReadRegionManager()
                               Returns the region manager.

Main_ManagerReadRtmRegionManager()
                               Returns the manager for the rtm package.

Main_ManagerReadStateManager() Returns the state manager.

Main_ManagerReadTypeManager()  Returns the type manager.

Main_ManagerSetMddManager()    Sets the mdd manager.

Main_ManagerSetModuleManager() Sets the module manager.

Main_ManagerSetRegionManager() Sets the region manager.

Main_ManagerSetRtmRegionManager()
                               Sets the rtm region manager.

Main_ManagerSetStateManager()  Sets the state manager.

Main_ManagerSetTypeManager()   Sets the variable type manager.

Main_MochaErrorPrint()         Prints error messages appropriately to Tcl
                               interface.

Main_MochaFilePrint()          Prints to file pointer fp

Main_MochaPrint()              Prints appropriately to either the Tcl
                               interface.

Main_Reinit()                  Reinitializes the main package.

Main_ReturnCreationTime()      Returns the creation time

Main_strConcat()               required

**********************************************************************

char *
Main_ConvertIntToString(
  int               i
)
  This function converts an integer (both positive and negative) into a
  string. It is the inverse of the library function atoi.

  Side Effects: User should free the string after use.

  Side Effects: Registers commands by modifying the TCL variable

int
Main_ReturnCreationTime(

)
  It returns the system-generated creation time. The system keeps an internal
  counter which increment once each time when this function is called. The
  creation time can be used to keep track of when the modules/atoms etc. are
  generated and it can be used for comparison.

  Side Effects: MainCreationTime is incremented.
